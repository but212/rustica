Rustica

Functional programming toolkit for Rust — type classes, persistent data structures, and more.

Home Benchmarks API Docs Crate GitHub

Overview

Rustica brings functional programming idioms and composable abstractions to Rust. It provides type classes (traits), immutable persistent data structures, and powerful optics for expressive, safe, and ergonomic code.

  • Type classes: Functor, Applicative, Monad, Monoid, IsoLens and more
  • Persistent data structures: PersistentVector (RRB-Tree), Validated for error accumulation
  • Functional utilities: State monad, Writer monad, composition tools
  • Optics: IsoLens for composable getters/setters
  • Interoperability with Rust's standard library

Quick Start

Add to your Cargo.toml:

[dependencies]
rustica = "0.7.0"  # or latest

Examples

Validated: Error Accumulation

use rustica::validated::Validated;
let valid: Validated<&str, i32> = Validated::valid(42);
let invalid = Validated::invalid("error");
let mapped = valid.fmap(|x| x * 2);
assert!(matches!(mapped, Validated::Valid(84)));

PersistentVector (pvec)

use rustica::pvec::{PersistentVector, pvec};
let v1: PersistentVector = pvec![1, 2, 3];
let v2 = v1.push_back(4);
assert_eq!(v2.get(3), Some(&4));

Functor, Applicative, Monad

use rustica::traits::functor::Functor;
let option = Some(2);
let doubled = option.fmap(|x| x * 2);
assert_eq!(doubled, Some(4));

IsoLens: Optics for Rust

use rustica::optics::{Iso, Lens};
struct Point { x: i32, y: i32 }
let lens = Lens::new(|p: &Point| p.x, |mut p: Point, x| { p.x = x; p });
let p = Point { x: 1, y: 2 };
let x = lens.get(&p);
let p2 = lens.set(p, 10);
