Overview

This page displays benchmark results for various functional programming abstractions implemented in Rustica. All benchmarks were run using Criterion.rs, a statistics-driven benchmarking library for Rust.

Benchmark Categories

Validated

Performance metrics for the Validated data type, which accumulates errors during validation.

  • Basic Creation and Access
  • Core Operations
  • Conversion Operations
  • Error Handling and Accumulation
  • Real-world Use Cases

State Monad

Performance metrics for the State monad, which handles stateful computations.

  • Basic Operations
  • Core Operations
  • Functor Operations
  • Applicative Operations
  • Monad Operations
  • Chain Operations
  • Real-world Use Cases

Other Monads

Performance metrics for other monadic types in Rustica.
