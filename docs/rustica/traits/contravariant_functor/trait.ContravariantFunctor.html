Expand description

A contravariant functor is a type constructor that allows mapping functions in a way that reverses their direction. While regular functors map functions forward (A -> B), contravariant functors map functions backward (B -> A).

fn contramap<U, F>(&self, f: F) -> Self::Output<U>
where F: Fn(&U) -> Self::Source, U: Clone,

Maps a function that transforms values of type U into values of type Self::Source, producing a new contravariant functor that works with type U.

This is the reference-based version of contramap, which only takes a reference to self.

  • U: The new input type for the resulting functor
