Traversing data structures with effects.

ยงTraversable

A trait for structures that can be traversed from left to right while preserving structure and accumulating effects in an applicative functor.

ยงMathematical Definition

For a type constructor T, a traversable instance consists of two operations:

  • traverse: (A -> F<B>) -> T<A> -> F<T<B>>
  • sequence: T<F<A>> -> F<T<A>>

where F is an applicative functor.
