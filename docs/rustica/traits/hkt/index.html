Higher-kinded type abstractions for generic programming. The HKT trait enables emulation of higher-kinded types in Rust.

Higher-kinded types (HKTs) are type constructors that take a type and return another type. Rust does not directly support higher-kinded types, but we can emulate them using associated types.

This module provides the HKT trait and related traits that form the foundation for higher-kinded polymorphism in the Rustica library.
