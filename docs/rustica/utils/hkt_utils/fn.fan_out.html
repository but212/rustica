Expand description

Applies multiple operations to a single input value.

This function takes a value and a collection of operations, applies each operation to the input value, and collects the results. This is useful when you need to transform a single input in multiple different ways.

ยงExamples

use rustica::utils::hkt_utils::fan_out;

// Single input value
let value = 10;

// Multiple operations to apply
let ops = vec![
    |n: i32| n + 5,
    |n: i32| n * 2,
    |n: i32| n * n
];

// Apply all operations to the input
let results = fan_out(value, ops);

// We get the result of each operation
assert_eq!(results, vec![15, 20, 100]);
